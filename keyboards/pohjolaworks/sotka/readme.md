# Sotka

![Sotka](https://i.imgur.com/hCCkpmP.jpg)

### Split Ortho keyboard

* Keyboard Maintainer: [ErkHal](https://github.com/ErkHal)
* Hardware Supported: Sotka rev 0.9 & 1.0
* Hardware Availability: [Source repo](https://github.com/pohjolaworks/sotka)

Make example for this keyboard (after setting up your build environment):

    make sotka:vial

Flashing example for this keyboard:

    make sotka:vial:flash
